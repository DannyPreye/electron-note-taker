## Getting Started

### Creating a New Note

Click the **New Note** button to create a fresh note. Give it a meaningful name to keep your notes organized.

### Writing with Markdown

You can use various markdown features:

1. **Headers** - Use `#` for headings
2. **Lists** - Both ordered and unordered
3. **Code blocks** - For snippets and examples
4. **Links** - Connect your ideas

### Code Example

Here's a simple JavaScript example:

```javascript
function greet(name) {
  console.log(`Hello, ${name}!`);
}

greet('World');
```

### Tips & Tricks

> **Pro Tip**: Use markdown shortcuts to speed up your note-taking!

- `**bold**` for **bold text**
- `*italic*` for *italic text*
- `` `code` `` for `inline code`
- `[link](url)` for [links](#)
